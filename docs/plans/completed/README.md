# Completed Implementation Plans

이 디렉토리에는 이미 구현이 끝난 `Implementation Plan` 문서를 보관한다.

정리 기준:

- 문서 본문에 `Status: Completed`가 명시된 구현 계획
- 또는 구현 완료가 문서 상단에 명확히 기록된 구현 계획

루트 `docs/plans/`에는 아래를 남긴다.

- 진행 중이거나 아직 실행되지 않은 구현 계획
- 설계 문서(`*-design.md`)
- 시나리오/스펙 참고 문서

이 규칙은 과거 계획의 실행 이력을 보존하면서도, 현재 진행 중인 계획 목록을 짧게 유지하기 위한 것이다.
